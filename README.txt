# Bubble PWA Starter

업로드 순서
1) `bubble_pwa_starter.zip`을 풀고, 폴더 안의 모든 파일을 GitHub 저장소 *루트*에 올립니다.
2) 사용자의 실제 게임 파일(예: `index_1.html`)을 `game.html`로 이름을 바꿔 같은 위치에 올립니다.
   - 또는 `index.html`의 iframe을 제거하고, 게임 코드를 직접 `index.html`에 넣어도 됩니다.
3) 저장소 Settings → Pages: Source = Deploy from a branch / Branch = main / Folder = /(root) → Save
4) `https://아이디.github.io/저장소명/` 접속 후 휴대폰에서 **홈 화면에 추가**.
   - Android(Chrome): ⋮ → 홈 화면에 추가 (최초 접속 후 약간 대기 시 설치 배너가 뜰 수도 있습니다.)
   - iPhone(Safari): 공유(□↑) → 홈 화면에 추가 (반드시 사파리에서)
